Для того чтобы кодировать/раскодировать строку нужно прописать:
node .\RLE.js code/decode (файл из которого будут кодировать/декодировать) (файл в который будут кодировать декодировать текст из первого файла)
input.txt-строка,которую нужно кодировать
decode.txt- строка,которую нужно раскодировать
output.txt-файл использовал для вывода кода/декода
А вообще код сразу выводит строку и закодированную/раскодированную строку.
